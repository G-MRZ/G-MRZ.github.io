# 命令
```sudo apt install [name]``` 安装应用

```sudo apt remove [name]``` 卸载应用

```sudo apt-get update``` 更新访问源列表

```sudo apt-get upgrade``` 更新已安装软件

```sudo update-alternatives --config editor``` 配置默认文本编辑器
